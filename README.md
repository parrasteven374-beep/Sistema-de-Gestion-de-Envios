# ABSTRACCION EJEMPLO
#SistemaEnviosPOO

Este proyecto fue desarrollado como práctica académica mientras estudio el técnico en desarrollo de software y aprendo conceptos de Programación Orientada a Objetos en C#, especialmente:

Abstracción
Herencia
Polimorfismo
Encapsulamiento

El sistema simula una empresa de logística que calcula costos y tiempos estimados de entrega utilizando diferentes tipos de vehículos como motos, camiones, bicicletas y furgonetas.

Actualmente soy estudiante de la Universidad Pontificia Bolivariana y este proyecto forma parte de mi proceso de aprendizaje en desarrollo de software.
